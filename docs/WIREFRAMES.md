# Wireframe Narrative & UX Flow (Week 3)

## 2.0 Student Portal
### 2.2 Job Board (Primary Search Interface)
* **Purpose**: The central tool for students to discover Information Systems-specific internships.
* **Primary Actions**: Search by keyword, filter by job type, and view details.
* **Components**: 
    * **Search Bar**: Keyword input with a "Search" button.
    * **Filter Sidebar**: Checkboxes for "Internship," "Co-op," and "Full-time".
    * **Job Cards**: Vertical list showing Title, Company, and Date Posted.
* **Accessibility**: All fields use `<label>` tags; high-contrast focus states.

## 3.0 Admin Portal
### 3.2 Job Management (CRUD Form)
* **Purpose**: Allow faculty to post new roles.
* **Components**: 
    * **Form Fields**: Job Title, Company, Description (Rich Text).
    * **Action Buttons**: "Save Draft" and "Publish".
